## <g-emoji class="g-emoji" alias="information_source" fallback-src="https://github.githubassets.com/images/icons/emoji/unicode/2139.png">ℹ️</g-emoji> CMI Translations / Locale

- Buy the [CMI](https://www.zrips.net/cmi/) premium plugin if you haven't already, and Install it on all your servers: <https://www.spigotmc.org/resources/3742/>
- Note: CMI requires the [CMI-Library](https://github.com/mrfdev/CMI/edit/master/Resources/FAQ/cmi-library.md) .jar, you can get it here: <https://www.spigotmc.org/resources/cmilib.87610/>

This is an early draft, I am still gathering information and facts, and waiting for Zrips to confirm a few things.

For those who need the info:

- CMI config.yml, set the country code there.
- CMILib config.yml, set the country code there.

Each have /translations/ folder with locale files ending with .yml, edit the country code you have previously set. If it doesn't exist, you can create your own file.

- Settings/ folder inside the plugins/cmi dir, also has locale files for items and deathmsgs. Make sure to enable using custom death msgs.
